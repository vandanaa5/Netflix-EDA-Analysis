# 🎬 Exploratory Data Analysis (EDA) Report - Netflix Dataset

# Dataset Overview
This dataset contains information about Movies and TV Shows available on Netflix, including details such as type, country, rating, release year, duration, and genre.

- Total Records: ~8800
- Key Features: type, country, rating, release_year, duration, listed_in

---

# Data Preparation
- Used previously cleaned dataset
- Missing values handled
- Date columns converted into proper format
- Created new features like:
  - year_added
  - duration_int
  - duration_type

---

# Exploratory Analysis

# 1. Content Type Distribution
- Movies are more prevalent than TV Shows on Netflix

# 2. Country-wise Content
- United States contributes the highest number of titles
- Content is globally distributed across multiple countries

# 3. Ratings Distribution
- TV-MA and TV-14 are the most common ratings
- Indicates focus on mature and teen audiences

# 4. Content Growth Over Time
- Significant increase in content after 2016
- Rapid expansion of Netflix library in recent years

# 5. Genre Analysis
- Drama and International content dominate the platform
- Wide diversity in genres available

# 6. Duration Analysis
- Movies typically range within a standard duration
- TV Shows vary based on number of seasons

---

# Key Insights

- Netflix has a higher number of Movies compared to TV Shows  
- The platform has expanded rapidly after 2016  
- United States leads in content production  
- Mature content (TV-MA) is highly dominant  
- International and Drama genres are most popular  

---

# Conclusion

The analysis highlights Netflix's growth strategy focused on expanding its content library globally, with a strong emphasis on diverse genres and audience segments. The insights derived can help understand viewing trends and content distribution strategies.
